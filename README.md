## 🏎️ MathSpeed Racing - Sitio Web Oficial

**Aprender nunca fue tan rápido y divertido**

---

### 📋 Descripción del Proyecto

MathSpeed Racing es una plataforma web educativa que presenta un juego innovador que combina carreras de autos de control remoto con desafíos matemáticos. El sitio web proporciona una experiencia inmersiva con diseño temático de carreras, animaciones dinámicas y toda la información sobre el producto.

---

### 🎯 Características Principales

#### 🎨 Diseño Profesional
- **Temática de Carreras**: Colores negro, plateado, azul y blanco
- **Tipografía Futurista**: Estilos modernos y letras amplias
- **Animaciones Dinámicas**: Autos en movimiento, semáforo de salida, banderas a cuadros
- **Diseño Responsivo**: Optimizado para desktop y móviles

#### 📱 Páginas Incluidas
1. **Inicio (index.html)** - Página de bienvenida con call-to-action
2. **Quiénes Somos (about.html)** - Misión, visión y valores
3. **El Juego (game.html)** - Descripción, reglas y mecánica
4. **Galería (gallery.html)** - Galería de fotos del prototipo
5. **Contacto (contact.html)** - Formulario de contacto e información

#### ✨ Elementos Visuales
- 🏎️ Animaciones de autos recorriendo la pantalla
- 🚦 Semáforo de salida interactivo
- 🏁 Banderas a cuadros ondeantes
- ⚡ Efectos de velocidad al cambiar secciones
- 📊 Contador de vueltas (LAP counter)

#### 📚 Contenido Educativo
- Suma de términos semejantes
- Resta de términos semejantes
- Multiplicación algebraica

#### 💰 Información de Precios
- Precio: $150.000 COP
- Incluye: Pista, autos, tarjetas matemáticas, cartas sorpresa y manual

### 🚀 Cómo Usar

#### 1. **Clonar el Repositorio**
```bash
git clone https://github.com/anittafernandez/mathspeed-racing-website.git
cd mathspeed-racing-website
```

#### 2. **Abrir Localmente**
- Abre `index.html` en tu navegador
- O usa un servidor local:
  ```bash
  python -m http.server 8000
  # Luego ve a http://localhost:8000
  ```

#### 3. **Personalizar**
- Reemplaza los colores en `styles.css` si deseas
- Agrega fotos reales en la carpeta `assets/`
- Actualiza la información de contacto en `contact.html`

### 🔧 Personalización

#### Cambiar Colores
Edita las variables CSS en `styles.css`:
```css
:root {
    --primary-dark: #1a1a1a;
    --primary-blue: #0066cc;
    --primary-silver: #c0c0c0;
    --accent-white: #ffffff;
}
```

#### Agregar Imágenes
```html
<img src="assets/tu-imagen.jpg" alt="Descripción">
```

#### Modificar Contenido
Edita directamente en los archivos HTML
